The Zen of Zen Garden

Zen Garden is a website dedicated to demonstrating the flexibility of the CSS design.
It allows you to download example files of both the HTML and the CSS or fork the site in Github. In the samples on the website,
the HTML does not change, only the CSS. Zen Garden invites users to practice and play with the CSS file. Users are invited to submit
their designs to Zen Garden. The site requests that the CSS used should be widely-supported elements only and that the user tests
their design across a variety of web browsers. For those users needing more information on how CSS works, the page offers a link to
CSS Resources. Zen Garden is interesting in much the way our last assignment was. The HTML stays the same while the CSS creates an
infinite amount possible designs.

The readings in Duckett focused on layout and overall design of websites in alignment with Zen Garden and its focus on how CSS can
create interesting and functional websites.

  • Positioning schemes such as absolute positioning can be seen in the fixed bottom elements
    • in the example Under the Sea by Eric Stoltz.
  • Use of a footer in A Walk in the Garden by Simon Van Hauwermeiren.
  • And the multitude of uses of borders, margins, and padding seen in all of the examples.

One of my favorite layouts was Apothecary by Trent Walton. I liked that there was an overall muted palette with bright splashes
of color in just a few locations. The bright color of the links made them easy to locate. I also liked the historic look of a
broadsheet or medicine label of the overall site and the vintage clipart.

In several places, the Apothecary layout takes advantage of the postion:absolute. One of these was after view all designs

.design-archives .previous a, .design-archives .next a {
position: absolute;
overflow: hidden;
width: 2em;
height: 1.25em;
color: #3f3d3a;
text-decoration: none;
white-space: nowrap;
text-indent: -9999px;

Additionally, the layout uses 30 instances of :after, which allows things to be inserted after an element. In Apothecary,
the creator used it for some of the graphics, for example, after the ul list which ends with translations, the CSS includes:

.zen-resources ul:after {
margin-top: 3em;
background: #bad1c2 url('img-halftone.png') repeat;
-webkit-transform: rotate(45deg) translateY(4.75em) translateX(9em);
-moz-transform: rotate(45deg) translateY(4.75em) translateX(9em);
-ms-transform: rotate(45deg) translateY(4.75em) translateX(9em);
-o-transform: rotate(45deg) translateY(4.75em) translateX(9em);
transform: rotate(45deg) translateY(4.75em) translateX(9em);
margin-bottom: -52.5em;
